# Чтобы запустить разработку выполни несколько шагов:

## ШАГ.1 открой один терминал и установи зависимости для клиента

```bash
cd client
npm i
npm run dev
```

## ШАГ.2 открой второй терминал и установи зависимости для сервера

```bash
cd server
npm i
npm run dev
```
